{0} leverages the advanced language capabilities of OpenAI's GPT-3 to assist in identifying potential vulnerabilities, generating reports, and performing traffic analysis.


This extension is designed to streamline the testing process and enhance the overall efficiency of security assessments. However, it is important to note that using this extension will result in requests being sent to the OpenAI API. Before using this extension, we advise checking OpenAI's privacy policy to ensure that it is acceptable for you to submit requests to their API.


For more information on how to use the {0} extension and to stay up-to-date with the latest developments, please visit the GitHub repository at https://github.com/aress31/burpgpt.
